CSS Background Properties

This page demonstrates the following CSS background properties:

  • background-color: Setting a solid color as background
  • background-image: Adding an image as background
  • background-attachment: Set to fixed so it doesn't scroll with content
  • background-size: Set to cover to fill the viewport
