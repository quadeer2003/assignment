CSS Comments Demonstration

This page demonstrates how to use comments in CSS. Open the page source to see the different ways comments are used in the stylesheet.

CSS comments are important for:

  • Documenting your code
  • Temporarily disabling CSS rules
  • Organizing sections of your stylesheet
  • Adding reminders and TODOs
  • Explaining complex selectors or calculations
