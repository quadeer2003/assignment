List Style Position

outside (Default)

  • This is the outside position. The bullet appears outside the block of text. If the text wraps to a new line, it will align with the text above, not with the bullet.
  • Another item with longer text to demonstrate wrapping behavior with the outside position.

inside

  • This is the inside position. The bullet appears inside the block of text. If the text wraps to a new line, it will align with the bullet.
  • Another item with longer text to demonstrate wrapping behavior with the inside position.

Shorthand (square inside)

  • This uses the list-style shorthand property to set both type and position.
  • Another item with longer text to demonstrate the shorthand property behavior.
