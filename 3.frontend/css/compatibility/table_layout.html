CSS Table Layout

table-layout: auto (Default)

With table-layout: auto, the column widths are determined by the content within them. The table takes into account all of the content before rendering the layout.

table-layout: fixed

With table-layout: fixed, the column widths are determined by the width of the table and the width of the columns in the first row. The content does not affect the column width.

Key Differences

  • table-layout: auto
    • The browser needs to examine all table content before determining column widths
    • Column widths are based on the content
    • Slower rendering for large tables
    • More responsive to content changes
  • table-layout: fixed
    • Only the first row is used to determine column widths
    • Column widths are based on column settings or are evenly distributed
    • Faster rendering for large tables
    • Better for controlled layouts
    • May truncate or wrap content that doesn't fit
