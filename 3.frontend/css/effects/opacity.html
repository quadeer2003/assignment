CSS Opacity

The opacity property specifies the transparency of an element. Values range from 0.0 (completely transparent) to 1.0 (completely opaque).

Basic Opacity Values

opacity: 1 (100%)

opacity: 0.75 (75%)

opacity: 0.5 (50%)

opacity: 0.25 (25%)

opacity: 0 (0%)

Important Notes About Opacity

  • When using opacity, all child elements inherit the same transparency.
  • If you want transparent backgrounds but fully opaque text, use rgba() colors instead of opacity.
  • Value range: 0.0 (completely transparent) to 1.0 (completely opaque).
  • Opacity affects the entire element, including borders and text.
