CSS Position Properties

Static Position

Static positioning is the default positioning for all elements. Elements are positioned according to the normal flow of the document.

Static

Relative Position

Relative positioning positions an element relative to its normal position. Setting top, right, bottom, and left properties will move the element away from its normal position.

Relative

This box is positioned 20px from the top and 20px from the left of its normal position.

Absolute Position

Absolute positioning positions an element relative to its nearest positioned ancestor (position is not static). If no positioned ancestors exist, it is relative to the document body.

Absolute

This box is positioned 150px from the top and 20px from the right of its closest positioned ancestor (the container).

Fixed Position

Fixed positioning positions an element relative to the viewport. The element will stay in the same place even if the page is scrolled.

Fixed

This box is positioned 20px from the bottom and 20px from the right of the viewport. Scroll to see it stay in place.

Sticky Position

Sticky positioning is a hybrid of relative and fixed positioning. The element is treated as relative until a scroll threshold is met, then it is treated as fixed.

Sticky

This box is positioned with top: 20px. Scroll to see it stick to the top when it reaches 20px from the top of the viewport.
