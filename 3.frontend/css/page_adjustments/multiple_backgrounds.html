Background Size Example

The following element has a single background with the 'contain' size property:

Common background-size values:

  • cover: Scales the image to cover the entire element (may be cropped)
  • contain: Scales the image to fit inside the element (may leave empty space)
  • auto: Uses the original image size
  • 100% 100%: Stretches the image to fill the element exactly
  • 200px 100px: Sets specific width and height
