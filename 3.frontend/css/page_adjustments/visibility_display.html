CSS Visibility and Display Properties

Visibility Properties

This element has visibility: visible. This is the default value and the element is visible.
This element has visibility: hidden. It takes up space but is not visible.

Note: Below is an element with visibility: collapse. For non-table elements, it works like hidden.

This element has visibility: collapse.
Normal row This is visible
Collapsed row This is collapsed (mainly for table rows)
Another normal row This is visible too

Display Properties

This element has display: none. It does not take up space and is not visible.
Before inline element
This element has display: inline. It only takes up as much width as necessary.
After inline element
This element has display: block. It takes up the full width available.
Before contents element -
This element has display: contents. The box itself is not generated, but its children are.
- After contents element
This element has display: list-item. It generates a box and a marker box (like a bullet).
